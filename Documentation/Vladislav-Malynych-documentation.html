Work Generalizing

My work was to create basic structure of the project, and implement a program that will get information(period of time) and will generate a new DocBook standartized file to be generated in an understandable form.

Resources

To make a project structure was created UseCase, Class and EntityRelation diagrams by the means of UML. For easy work with files was used dom4j library.

Implementation

At first was created a model of the project, and interfaces according to it. Then was created a classes for generating xml and it's converting, also it includes compatibility features. Finally was done a web page for generating PDF file.
